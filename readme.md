# MvvmLogin — מערכת התחברות והרשמה ב-MVVM

פרויקט לימודי זה מציג מימוש של דף התחברות והרשמה בטכנולוגיית **.NET MAUI** (מבוסס .NET 9) תוך שימוש בתבנית העיצוב **MVVM (Model-View-ViewModel)**. 

המערכת מעוצבת ומותאמת אישית עבור מוסד החינוך "גני אורנים" ומדגימה כיצד לבצע הפרדה מלאה בין לוגיקת האפליקציה לבין ממשק המשתמש (UI).

---

## 🏗️ מבנה הפרויקט

הפרויקט בנוי לפי עקרונות MVVM ומחולק לתיקיות הבאות:

*   **`Models/` (המודל):**
    *   `User.cs` - מייצג את ישות המשתמש במערכת (שם משתמש, אימייל, סיסמה ותמונת פרופיל).
    *   `ObservableObject.cs` - מחלקה המממשת את ממשק `INotifyPropertyChanged` המאפשרת הודעה על שינויי ערכים במאפיינים.
*   **`ViewModels/` (לוגיקת התצוגה):**
    *   `ViewModelBase.cs` - מחלקת בסיס לכל ה-ViewModels, יורשת מ-`ObservableObject`.
    *   `LoginPageViewModel.cs` - מנהל את המצב והלוגיקה של דף ההתחברות (קלט משתמש, הודעות שגיאה, אימות מול השירות וקישור לפקודת התחברות).
*   **`Pages/` (התצוגה - View):**
    *   `LoginPage.xaml` / `.xaml.cs` - דף ההתחברות למערכת המקושר ל-ViewModel שלו דרך ה-`BindingContext`.
    *   `CreateAccountPage.xaml` / `.xaml.cs` - דף הרשמה ויצירת חשבון חדש במערכת (מיועד להשלמה על ידי התלמיד).
*   **`Services/` (שירותים):**
    *   `MockService.cs` - שירות אימות מדומה (`MockAuthService`) המדמה מסד נתונים בזיכרון הסטטי של האפליקציה לצורך בדיקת חיבור משתמשים ורישום משתמש חדש.

---

## 🛠️ טכנולוגיות וכלים

*   **.NET MAUI (.NET 9):** פלטפורמה לפיתוח אפליקציות Cross-Platform (Windows, Android, iOS, macOS).
*   **RTL (Right-To-Left):** תמיכה מלאה בכתיבה מימין לשמאל עבור קהל יעד עברי.
*   **Compiled Bindings (x:DataType):** שימוש בקישורים מקומפלים לשיפור ביצועי האפליקציה ומניעת שגיאות הקלדה בזמן הידור.
*   **Commands (ICommand):** שימוש בפקודות לקישור אירועי לחיצה של כפתורים ללוגיקה ב-ViewModel ללא צורך בכתיבת קוד ב-Code-Behind.

## 🚀 הוראות הרצה

כדי להריץ את הפרויקט במחשב המקומי שלך:

1.  וודא שמותקנת גרסת **Visual Studio 2022** עם חבילת הפיתוח של **.NET MAUI** (תומכת .NET 9).
2.  פתח את קובץ ה-Solution: `MvvmLogin.sln`.
3.  בחר את פלטפורמת היעד (לדוגמה: Windows Machine או אמולטור Android).
4.  הרץ את האפליקציה (לחץ על F5).
